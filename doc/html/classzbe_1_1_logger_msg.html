Detailed Description

Wrapper class to use the operator "<<" in the log messages.

This class is not used directly.

This class let us use the operator "<<" to concatenate distinct type of variables. i.e.: 

  int a = 3;
  double b = 3.1415;
  bool c = true;
  ZBE_LOG_INFO("A message with " << a << " variables ( " << b << ") of distinct type: " << c);

[INFO]> A message with 3 variables (3.1415) of distinct type: true;

This class also change the printed value of a bool variable from 1 or 0 to "true" or "false.

To use a custom type in the message composition, the new type should define an operator<< that returns a stringstream compatible type, i.e. a std::string.
